# TikTok Shop Automation Suite

Application Next.js qui automatise la production de contenus pour TikTok Shop :

- Centralisation du catalogue produits et suivi de pipeline (statut + priorité)
- Générateur de scripts courts prêts à tourner avec hooks, CTA et B-roll
- Planification automatique des créneaux de publication selon les heures fortes TikTok
- Playbook pour industrialiser la mise en avant vitrine

## Démarrage

```bash
npm install
npm run dev
```

## Production

```bash
npm run build
npm start
```

Le déploiement sur Vercel est prêt via `vercel deploy --prod`.
